Bubbler Totem

    
Control the totem directly over Bluetooth - no WiFi required.

    
    

    
On iPhone, Safari can't do this - use
      Bluefy
      or WebBLE instead. Desktop/Android Chrome and Edge work directly.

    
Tap Debug (top right) to see a live log of every BLE command/response, including raw
      bytes - tap Copy log there any time to grab it for troubleshooting.
